Roll Dice
=========

A small library that rolls a classic dice : 1 to 6.

## Installation

  `npm install @bdavidxyz/rolldice`

## Usage

    var rollDice = require('@bdavidxyz/rolldice');

    var rolled = rollDice();

    console.log(rolled);
  
  
  Output should be a number between 1 (inclusive) and 6 (inclusive)
